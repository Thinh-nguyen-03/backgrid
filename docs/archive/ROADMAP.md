# Roadmap

## MVP
- API + workers + metrics service
- Parquet cache, HRP optimizer
- Dash dashboard, core observability

## Beta
- Multi‑user tenancy
- Paper‑trading mode (Alpaca)
- Strategy registry UI
- Full CI/CD with canary deploys

## GA
- Broker adapters (IBKR/Alpaca/FIX) behind interfaces
- Advanced risk (VaR/CVaR), optimizer constraints
- Multi‑cluster scale‑out and regional replicas
