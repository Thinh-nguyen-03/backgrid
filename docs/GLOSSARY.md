# Glossary

- **Backtest** — Simulate a strategy over historical data.
- **HRP (Hierarchical Risk Parity)** — Portfolio method that clusters assets by correlation.
- **Sharpe / Sortino** — Risk‑adjusted return metrics.
- **Max Drawdown** — Largest peak‑to‑trough equity decline.
- **Hypertable** — TimescaleDB partitioned table for time‑series.
- **Idempotent** — Safe to retry without duplicate side‑effects.
- **p95/p99** — 95th/99th percentile latency metrics.
